# Quiz Game

## Descripcion
<p>Quiz Game como su nombre lo indica es un juego de preguntas, en el cual tenemos que seleccionar la respuesta correcta
entre las distintas preguntas, este juego esta hecho en consola y cuanta con algunas funcionalidades que lo hacen divertido
(tomando en cuenta que las preguntas no son dinamicas lo cual indica que siempre que lo juegues saldran las mismas preguntas).</p>

## Funcionalidades
Entre sus funcionalidades mas fuertes tenemos su **multijugador** una funcion la cual nos permite jugar con nuestros amigos ya que
el juego tiene una opcion de jugar de nuevo y gracias a esta podemos invitar a nuestros amigos a superar nuestro puntaje, y aqui esta
otra de sus funciones y es que el juego cuenta con un **Score de puntuacion** con el cual podemos ver cual fue la puntuacion y 
compararla con la nuestros amigos y podemos ver quien tuvo mejor puntuacion y si queremos tambien podemos mejorar nuestro record.
